Project Description.

This Data Structures and Algorithms portfolio showcases the implementation of fundamental queue-based data structures using linked lists in a Flask web application. The project features two core components: a standard Queue with enqueue and dequeue operations, and a Double-Ended Queue (Deque) that supports insertion and deletion at both ends through enqueue, enqueue head, dequeue, and dequeue head methods. Built with Python Flask and integrated with an interactive user interface, this portfolio demonstrates practical applications of linear data structures. Each implementation follows best practices in data structure design, providing users with hands-on visualization of queue operations and their real-world applications.
